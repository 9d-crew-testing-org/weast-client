# WEAST Client

Client for connecting and retrieving data from the WEAST Intellistar 2 data server. 

---

## 📦 Building for i2XD and i2JR

### Requirements

- A computer or virtual machine running **Windows 7 (64-bit)**
- **Python 3.8.10**
- **PyInstaller**

### Build Instructions

1. Download this repository as a `.zip` file  
2. Extract the contents to a folder
3. Install pip packages `requests` and `paho-mqtt`
4. Run `build-encoder.bat` 

The compiled executable will be generated in the `dist/` directory.
